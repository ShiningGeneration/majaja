# Majaja

## Prerequisites

+ nodejs@0.12
+ npm@2
+ gulp@3.8

## Installation

Install packages / libraries from NPM.

```bash
$ npm install
```

Build with Gulp

```bash
$ gulp
````

Rock with live-reload

```bash
$ gulp server
```

Open browser with [http://localhost:8080/webpack-dev-server/index.html](http://localhost:8080/webpack-dev-server/index.html)
